## 原理分析

通过 utf-8 将操作的 js 代码转换为一串数字，然后每隔两个字符中添加一个字母，这一段字符串就是需要解密的程序。

## 字符表提取程序

```Python
def decrypt_string(encrypted_string):
	decrypted = ""
	num_str = ""
	
	for char in encrypted_string:
		if char.isalpha():
			if num_str:
				num = int(num_str)
				if num <= 128:
					decrypted += chr(num) 
				else:
					decrypted += str(num)
				num_str = ""
		else:
			num_str += char
			
	return decrypted

decrypted_string = decrypt_string(encode_text)
```
